# nginx-brotli-mysql-phpmyadmin
Lemp NGINX

## Автоматическая синхронизация с GitHub

Этот репозиторий настроен с автоматической синхронизацией изменений с GitHub.

### Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что git настроен и подключен к репозиторию:
```bash
git remote -v
```

Если нужно добавить remote:
```bash
git remote add origin https://github.com/totalnewera-boop/nginx-brotli-mysql-phpmyadmin.git
```

### Использование

Запустите скрипт автоматической синхронизации:
```bash
python auto_sync.py
```

Скрипт будет:
- Отслеживать все изменения файлов в директории
- Автоматически добавлять изменения в git
- Создавать коммиты с временной меткой
- Отправлять изменения в GitHub через 5 секунд после изменения

### Остановка

Нажмите `Ctrl+C` для остановки синхронизации.

### Примечания

- Скрипт игнорирует системные файлы (`.git`, `__pycache__`, `.pyc` и т.д.)
- Изменения группируются - коммит создается через 5 секунд после последнего изменения
- Убедитесь, что у вас настроена аутентификация GitHub (Personal Access Token или SSH ключи)
